- **Website:** scamadviser.com

- **Revenue:** Estimated $10-15M ARR
- **Strengths:** Massive user base, established brand, mobile app, B2B API already selling, global coverage
- **Weaknesses vs us:** Less Canadian-specific data, no interactive education/quizzes, no crypto wallet analysis, no XRPL support

### Trend Micro ScamCheck
- **Backed by:** Trend Micro ($1.8B revenue company, publicly traded)
- **What they do:** AI-powered scam detection â€” text, links, screenshots, call analysis

- **Strengths:** Massive R&D budget, existing security suite integration, global brand recognition, enterprise relationships
- **Weaknesses vs us:** Character limits on input, struggled with screenshots, no country-specific guidance, no education modules

### Norton Genie (now part of LifeLock)
- **Backed by:** Gen Digital (Norton parent, $3.4B revenue)
- **What they do:** AI scam detector â€” paste text or upload screenshots, follow-up questions for context

### McAfee Scam Detector
- **Backed by:** McAfee ($2B+ revenue)
- **What they do:** AI-powered scam text analysis
- **Testing results (KSL):** Most reliable overall in KSL testing
- **Limitation:** No screenshot support, copy-paste only

- **Weaknesses vs us:** No screenshot support, no country-specific guidance, no education, US-focused

### Bitdefender Scamio
- **Backed by:** Bitdefender ($600M+ revenue)
- **What they do:** AI chatbot for scam analysis, chat-based interface
- **Testing results (KSL):** Hedged answers ("likely a scam" vs definitive), struggled with complex emotional/romance scams
- **Strengths:** Conversational UI, part of Bitdefender security suite
- **Weaknesses vs us:** Indecisive outputs, no country-specific guidance, no education

## Tier 4: Free/Government Tools

### CAFC (Canadian Anti-Fraud Centre)
- **What they do:** Fraud reporting, static advisories, phone line (1-888-495-8501)
- **Strengths:** Government authority, trusted, bilingual
- **Weaknesses:** No real-time detection, no interactive tools, PDF-based education, no API

### BBB Scam Tracker
- **What they do:** Scam reporting database, search by keyword/type/location
- **Strengths:** Trusted brand, crowdsourced data, US + Canada coverage
- **Weaknesses:** Reporting tool not detection tool, no real-time analysis

### Have I Been Pwned
- **What they do:** Email/password breach checking, notification service
- **Users:** Billions of records, widely trusted
- **Strengths:** Free, authoritative, API available
- **Weaknesses:** Email breaches only â€” no text/URL/phone/crypto analysis, no education

### Google Safe Browsing
- **What they do:** URL blocklist used by Chrome, Firefox, Safari
- **Strengths:** Largest coverage, built into every major browser
- **Weaknesses:** Binary safe/unsafe, no explanation, no education, lag time on new threats
